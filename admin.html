📌 如何更新到網路上？

在此頁面編輯完成後，點「⬇️ 下載 data.json」，將下載的 data.json 上傳到你的 GitHub repository 根目錄，替換原本的檔案即可。GitHub Pages 會在幾分鐘內自動更新網站。
